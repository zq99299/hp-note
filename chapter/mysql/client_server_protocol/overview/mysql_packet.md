# MySqls数据包

单包最大长度16M(2^24 -1 byte); 因为由以下包头组成; 最大长度其实就是3个byte组成的无符号最大值

| 类型 |	名称 | 描述
|-----|-----|----
| `INT<3>`	| payload_length | 有效载荷的长度。数据包中的字节数超出组成数据包标题的起始4个字节。
| `INT<1>`	| sequence_id	| 序列号
| `STRING<VAR>`	| payload	| [len = payload_length ]数据包的有效载荷
